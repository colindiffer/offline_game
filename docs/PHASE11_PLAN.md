# Phase 11: Memory, Vocabulary & Strategy

## Overview
Phase 11 expands the collection to 24 games, introducing a popular solitaire variant, a memory-focused challenge, and a word-logic game.

## New Games

### 1. Memory Match
- **Goal**: Find and match all pairs of face-down cards.
- **Visuals**: Premium card backs, smooth flipping animations, and satisfying "match" effects.
- **Controls**: Tap to flip cards.
- **Progression**: Larger grids (from 4x3 up to 8x6) and shorter time limits as levels increase.

### 2. Word Guess
- **Goal**: Guess the hidden 5-letter word in 6 attempts.
- **Visuals**: Modern minimalist grid with high-contrast color feedback (Green/Yellow/Grey).
- **Controls**: Stylized on-screen keyboard.
- **Progression**: Levels progress from common 5-letter words to rare and challenging vocabulary.

### 3. Spider Solitaire
- **Goal**: Order all cards in descending runs from King to Ace by suit.
- **Visuals**: Utilizes the overhauled full-screen solitaire layout with large, readable cards.
- **Controls**: Drag-and-drop or tap-to-move stacking.
- **Progression**: Difficulty levels control the number of suits used (1, 2, or 4 suits).

## Implementation Rules
- All new games will feature **Dynamic Scaling** to fill the screen width.
- All will include **Level-Based Progression** with persistent storage.
- Full-screen **How to Play** tutorials will be implemented for each.

## UI Refinements (Mobile Optimization)
- **Spider Solitaire**: Maximized 10-column width and increased vertical spread.
- **Memory Match**: Standardized card shapes and fixed flip visibility.
- **Word Guess**: Resolved header overlap and improved keyboard positioning.
