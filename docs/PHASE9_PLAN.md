# Phase 9: Logic & Memory Expansion

## Overview
Phase 9 adds 3 new games focused on memory, logic, and tile-matching, bringing the total game count to 21.

## New Games

### 1. Mahjong Solitaire
- **Goal**: Match pairs of identical tiles to clear the board.
- **Visuals**: Layered "3D" tile stacks with traditional or modern symbols.
- **Controls**: Tap to select tiles (only "free" tiles can be picked).
- **Progression**: Increasing complexity of tile layouts and more layers.

### 2. Hangman
- **Goal**: Guess the hidden word before the "hangman" is fully drawn.
- **Visuals**: A premium animated hangman drawing and a custom stylized keyboard.
- **Controls**: Tap letters on an on-screen keyboard.
- **Progression**: Word complexity increases; hints become more limited.

### 3. Simon Says (Memory Pattern)
- **Goal**: Repeat a growing sequence of lights and tones.
- **Visuals**: Four large, glowing colored pads with haptic feedback and distinct sounds.
- **Controls**: Tap the pads in the correct order.
- **Progression**: Sequences get longer and flash faster as you advance levels.
